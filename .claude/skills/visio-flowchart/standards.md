# 公司流程图规范配置

> 在此文件中覆盖 ISO 5807 默认规范。**只填写需要覆盖的项**，留空的项将自动使用 ISO 5807 默认值。
>
> - 颜色格式：`#RRGGBB`（6 位十六进制，如 `#1E90FF`）
> - 字体名称：需为当前系统已安装的字体（如 `微软雅黑`、`宋体`、`Arial`）
> - 形状类型：`圆角矩形` / `矩形` / `菱形` / `平行四边形` / `双边框矩形`
> - 字号格式：数字 + `pt`（如 `11pt`）
