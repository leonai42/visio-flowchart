# 设计调整说明

> 版本：v1.0
> 创建日期：2026-05-15
> 对应 Phase 2 原始设计：design.md

## 调整清单

### 1. 泳道间距微调

- **原始设计**（design.md Risks 表）：泳道间距 2 英寸
- **实际实现**（SKILL.md 4.2 + 泳道图规则）：泳道间距 2.5 英寸
- **原因**：2.5 英寸在 Visio 中视觉效果更好，节点间距 1.5 英寸 + 泳道间距 2.5 英寸的比例更协调
- **影响**：仅影响 Visio 布局参数，不影响 Skill 行为语义

### 2. 前置检查增加 uv 可用性验证

- **原始设计**：仅检查 Visio 运行状态和 visio-mcp 安装
- **实际实现**：增加步骤 0 —— 检查 `uv` 命令可用性
- **原因**：Review 发现 .mcp.json 使用 `uv` 启动 visio-mcp，uv 不可用会先于 visio-mcp 报错
- **影响**：改善错误提示的精确度和友好性

### 3. 异常处理模板重设计

- **原始 spec**：节点清单为线性列表
- **实际实现**：重新设计为更清晰的垂直分支结构
- **原因**：Review 发现原 ASCII 图中 `└─` 缩进歧义，改为 `→` 和 `↓` 明确方向
- **影响**：仅影响模板展示，不影响 Skill 核心逻辑

### 4. MCP 编排增加错误处理

- **原始设计**：5 步调用链无错误处理
- **实际实现**：每步增加失败提示和停止/跳过策略
- **原因**：Review 发现缺少错误处理会导致级联失败
- **影响**：增强 Skill 鲁棒性，不影响正常流程
